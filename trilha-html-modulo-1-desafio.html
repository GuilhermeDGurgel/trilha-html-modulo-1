O que é Html?

     HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas da web. Ela serve para estruturar e organizar o conteúdo que aparece nos navegadores, como textos, imagens, vídeos, links e outros elementos visuais. Apesar de ser chamada de "linguagem", o HTML não é uma linguagem de programação, e sim de marcação. Isso significa que ele não realiza cálculos ou ações lógicas, como um programa faria, mas sim marca e define partes do conteúdo para que o navegador saiba como exibi-las. Por exemplo, com HTML é possível indicar que um trecho de texto é um título, outro é um parágrafo, ou que determinada imagem deve aparecer em um ponto da tela. Essas instruções são feitas usando tags (ou etiquetas), como h1, p, img, entre muitas outras. HTML é a base da web e, junto com o CSS (que cuida do estilo) e o JavaScript (que cuida da interatividade), forma o trio essencial para desenvolvimento de sites.

Como Surgiu o Html?

     O HTML surgiu no início da década de 1990 como parte de um projeto para facilitar o compartilhamento de informações entre cientistas ao redor do mundo. Seu criador foi Tim Berners-Lee, um físico britânico que trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear), na Suíça. Naquela época, os pesquisadores enfrentavam dificuldades para acessar e organizar dados disponíveis em computadores diferentes. Para resolver esse problema, Tim Berners-Lee propôs um sistema baseado em hipertexto — um formato que permite conectar documentos entre si através de links. Assim, em 1991, ele desenvolveu os primeiros componentes da World Wide Web (WWW), incluindo: O HTML (HyperText Markup Language) — a linguagem de marcação para estruturar os documentos; O HTTP (HyperText Transfer Protocol) — o protocolo para transferir esses documentos; E o primeiro navegador web, chamado WorldWideWeb. A primeira versão do HTML era bastante simples e contava com cerca de 20 tags básicas, como p para parágrafo e a para link. Com o tempo, o HTML evoluiu para suportar uma variedade cada vez maior de elementos e funções, acompanhando o crescimento da internet. Hoje, o HTML é mantido por organizações como o W3C (World Wide Web Consortium), e sua versão mais atual é o HTML5, que trouxe novas funcionalidades, como vídeos incorporados, elementos semânticos e melhorias para dispositivos móveis.

Tags Mais Utilizadas (blogs)

Títulos e Parágrafos
  • <h1> até <h6> : Representam títulos, sendo <h1> o mais importante e <h6> o menos importante.
  • <p> : Define um parágrafo de texto.
Formatação de Texto
  • <mark> : Destaca o texto com um fundo amarelo (como um marca-texto).
  • <small> : Deixa o texto menor. (small)
  • <i> : Deixa o texto em itálico (ênfase leve ou estilo).
  • <u> : Sublinha o texto. (sublinhado)
  • <strong> : Dá ênfase forte ao texto (semântica de importância, geralmente em negrito).
  • <del> : Mostra que o texto foi deletado (riscado).
  • <sub> : Deixa o texto em subscrito (abaixo da linha, usado em fórmulas químicas, por exemplo: H₂O).
  • <sup> : Deixa o texto em sobrescrito (acima da linha, usado em potências: 2⁴).
Listas
  • <ol> : Cria uma lista ordenada (numerada).
  • ><ul>
  • <li> : Define cada item de uma lista (ol ou ul).
Links e Linhas
  • <a> : Cria um link para outra página ou site. Exemplo
  • <hr> : Insere uma linha horizontal (divisória).
Outras Tags
  • <blockquote> : Destaca uma citação longa, geralmente com recuo.
    • Exemplo de blockquote
  • <font> : Antiga tag usada para mudar fonte, tamanho ou cor do texto (obsoleta no HTML5, use CSS).
  • <abbr> : Mostra uma abreviação e permite indicar seu significado com um título (tooltip).
